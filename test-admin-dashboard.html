การทดสอบการแสดงข้อมูลผู้ใช้ในหน้า Admin Dashboard

  1. เข้าสู่ระบบที่ http://localhost:3000/login-portal
  2. ไปที่หน้า Admin Portal: http://localhost:3000/admin-portal/dashboard
  3. ตรวจสอบว่าข้อมูลผู้ใช้ที่ login เข้ามาแสดงถูกต้องตามรูปแบบด้านล่าง
  4. ทดสอบปุ่ม "จัดการคำขอ" ว่าสามารถนำไปยังหน้าจัดการคำขอได้
  5. ทดสอบปุ่ม "จัดการผู้ใช้งาน" ว่าสามารถนำไปยังหน้าจัดการผู้ใช้ได้

แดชบอร์ดผู้ดูแลระบบ

ยินดีต้อนรับ, [ชื่อผู้ใช้]

ตำแหน่ง: [ตำแหน่งเป็นภาษาไทย] อีเมล: [อีเมลผู้ใช้]

การเปลี่ยนแปลงที่เกิดขึ้น:

  • เพิ่มการแสดงข้อมูลผู้ใช้ในส่วน welcome message ของหน้า admin dashboard
  • แสดงชื่อผู้ใช้, ตำแหน่ง (เป็นภาษาไทย), และอีเมล
  • เพิ่ม avatar ที่แสดงตัวอักษรแรกของชื่อผู้ใช้
  • ปรับปรุงการแสดงผลใน sidebar ของ AdminLayout ให้สอดคล้องกัน
  • เพิ่มปุ่ม "จัดการคำขอ" ในหน้า dashboard เพื่อให้เข้าถึงหน้าจัดการคำขอได้ง่ายขึ้น
  • แก้ไขปัญหาการทำงานของปุ่ม "ดำเนินการ" และ "ตรวจสอบ" ให้กรองสถานะคำขอได้ถูกต้อง
  • เพิ่ม API endpoint สำหรับดึงข้อมูล admin users ใน backend

หน้าที่สามารถเข้าถึงได้:

  • Admin Dashboard - หน้าแรกของผู้ดูแลระบบ
  • จัดการคำขอ - หน้าจัดการคำขอทั้งหมด
  • จัดการผู้ใช้งาน - หน้าจัดการผู้ใช้งานแอดมิน
